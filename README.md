# Poker Tools

德州扑克学习工具，纯前端单文件，无后端无依赖。

## 在线访问

- **胜率计算器**：[index.html](./index.html) — 输入底牌、公共牌、位置和对手数，蒙特卡洛模拟胜率并给出行动建议；自带规则页和位置详解
- **9-max 起手牌矩阵**：[range-chart.html](./range-chart.html) — 小额现金桌的 13×13 起手牌范围矩阵，9 个位置可切换

## 功能

### 胜率计算器 (`index.html`)

- 7 个可点击牌槽：2 张底牌 + 0/3/4/5 张公共牌
- 52 张扑克牌点选界面，已使用的自动置灰
- 9 种位置选择（UTG / UTG+1 / MP / LJ / HJ / CO / BTN / SB / BB）
- 1–8 个对手数量，对手范围支持紧/标准/松/随机
- 蒙特卡洛 4000–10000 次模拟，输出胜率/平局/败率/权益
- 根据权益、阶段、位置、对手数动态生成行动建议（加注/价值下注/控池/听牌/弃牌）
- 内置德扑规则速览和位置详解（带圆桌示意图）
- 响应式设计，桌面端 13 列点牌区，手机端切换为 4 列纵排，支持 iOS 安全区域

### 起手牌矩阵 (`range-chart.html`)

- 9 个位置一键切换，深红/橙/灰三色显示加注/3-bet 跟/弃牌
- 实时显示当前位置范围占比（按 1326 组合数加权）

## 技术栈

纯 HTML + CSS + JavaScript，无构建步骤，无外部依赖。直接在浏览器打开 HTML 即可运行。

7 张牌的最大牌型评估器是手写实现，完整覆盖皇家同花顺到高牌的所有十种牌型。

## 校验示例

打开胜率计算器可以验证以下经典数值：

| 场景 | 预期胜率 |
| --- | --- |
| AA vs 1 个随机对手 | ~85% |
| AA vs 8 个随机对手（满桌全跟） | ~31% |
| 72o vs 1 个随机对手 | ~35% |
| AKs vs 紧手范围（含 AA/KK） | ~45% |

## 本地运行

直接双击 HTML 文件即可，或：

```bash
python3 -m http.server 8000
# 浏览器打开 http://localhost:8000
```
